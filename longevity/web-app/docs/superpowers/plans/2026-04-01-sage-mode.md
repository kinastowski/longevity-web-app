# SAGE Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a global SAGE Mode toggle that shifts all 6 experts from giving recommendations to asking deep wisdom-eliciting questions, saving user responses as LegacyFragments in DynamoDB.

**Architecture:** `sage_mode: boolean` added to UserProfile. The shared conversation handler reads it and branches the system prompt. profileExtractor receives a `sageMode` flag and writes a LegacyFragment instead of extracting profile facts. Frontend toggle calls `client.models.UserProfile.update()` directly (no custom mutation). Global nav bar in `layout.tsx` renders the toggle on every page.

**Tech Stack:** AWS Amplify Gen 2, AppSync (GraphQL), DynamoDB, Lambda (Node.js/TypeScript), Next.js 14, shadcn/ui Switch, Vitest

```typescript
'use client';

import { useState, useEffect } from 'react';
import { generateClient } from 'aws-amplify/api';
import type { Schema } from '@/amplify/data/resource';
import { Switch } from '@/components/ui/switch';
import { Label } from '@/components/ui/label';

const client = generateClient<Schema>({ authMode: 'userPool' });

export function SageToggle() {
  const [profileId, setProfileId] = useState<string | null>(null);
  const [sageMode, setSageMode] = useState(false);

  useEffect(() => {
    void client.models.UserProfile.list().then(({ data }) => {
      const profile = data[0];
      if (profile) {
        setProfileId(profile.id);
        setSageMode(profile.sage_mode ?? false);
      }
    });
  }, []);

  const toggle = async () => {
    if (!profileId) return;
    const next = !sageMode;
    setSageMode(next); // optimistic update
    await client.models.UserProfile.update({ id: profileId, sage_mode: next });
  };

  return (
    <div className="flex items-center gap-2">
      <Switch id="sage-mode" checked={sageMode} onCheckedChange={toggle} />
      <Label
        htmlFor="sage-mode"
        className="text-sm font-medium cursor-pointer select-none text-slate-700"
      >
        SAGE Mode
      </Label>
    </div>
  );
}
```
